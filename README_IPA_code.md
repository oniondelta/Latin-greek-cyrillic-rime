# 拉丁 洋蔥形碼 IPA國際音標 編碼

- 「調出符」以 en_wel.schema.yaml ✪EuropeLetter-Onion✪ 為範例，其他方案編碼一樣，但「調出符」會不同。

- 開頭「 ; 」為「基本調出符」

- 開頭「 ;; 」為「大寫調出符」

- 開頭「 ' 」「 ; 」為「組合字符 “ Combining Diacritical Marks”調出符」

- 以下標示：前為 IPA 字符， ( … ) 內為編碼。
